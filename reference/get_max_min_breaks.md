# get max and min deaths_averted from results

get max and min deaths_averted from results

## Usage

``` r
get_max_min_breaks(result)
```

## Arguments

- result:

  output of
  [`load_sensitivity_analysis_data()`](https://sempwn.github.io/pss-model-comparison/reference/load_sensitivity_analysis_data.md)

## Value

vector of length 2 representing min and max
