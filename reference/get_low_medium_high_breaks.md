# get result breaks into small, medium, and large categories

get result breaks into small, medium, and large categories

## Usage

``` r
get_low_medium_high_breaks(result)
```

## Arguments

- result:

  output of
  [`load_sensitivity_analysis_data()`](https://sempwn.github.io/pss-model-comparison/reference/load_sensitivity_analysis_data.md)

## Value

vector of length 4 representing the 0th 33rd, 66th and 100th percentiles
